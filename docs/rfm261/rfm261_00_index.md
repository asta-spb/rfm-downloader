## Структура имени файла ФЭС

```
FMTI_V_N_GGGGMMDD_TIXXXPPPPP.xml
```

- `FMTI` — идентификатор формы (FM01–FM04)
- `V` — код вида информации (1–7)
- `N` — ИНН+КПП (19 знаков) или ИНН ИП (12 знаков)
- `GGGGMMDD` — дата направления
- `TIXXXPPPPP` — порядковый номер (TI = тип, XXX = код филиала или 000, PPPPP = номер)

Подробнее: [раздел 3.1 и 3.2](rfm261_04a_naimenovanie.md)

## Ключевые идентификаторы

**ИдентификаторЗаписи** (тег `НомерЗаписи`) — уникален на весь жизненный цикл записи:
```
<год>_<ИНН>_<КПП>_<порядковыйНомер>
```
Пример: `2015_7702345025_000000000_00000001`

Подробнее: [таблица 2.6](rfm261_03b_tipy_lico.md)
