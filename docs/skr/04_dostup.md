# 4. Получение доступа к методам

## 4.1. Порядок получения доступа к тестовым методам

Для получения доступа к тестовым методам необходимо через свой Личный кабинет на официальном сайте Росфинмониторинга направить заявку в техническую поддержку:

- **Тематика обращения:** «Сервисный концентратор»
- **Текст:** «Регламентная процедура. Получение доступа к тестовым методам ЭС»

---

## 4.2. Порядок получения доступа к продуктивным методам

1. Успешно пройти тестирование, вызвав **все тестовые методы ЭС**;
2. Заполнить [форму заявки](appendix_1_zayavka.md) на получение доступа и вместе с неформатированными конвертами запросов-ответов (только те, которые содержат параметры) к тестовым методам в формате JSON упаковать в zip-архив.  
   > **Прикладывать конверт запрос-ответа метода `authenticate` не требуется.**
3. Направить архив в Росфинмониторинг через заявку в техническую поддержку:
   - **Тематика обращения:** «Сервисный концентратор»
   - **Текст:** «Регламентная процедура. Получение доступа к продуктивным методам ЭС»
