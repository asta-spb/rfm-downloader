## 1.1. Описание электронного сервиса

Электронный сервис «Сервисный концентратор Росфинмониторинга» предоставляет возможность пользователям Личного кабинета на официальном сайте Росфинмониторинга направлять в Федеральную службу по финансовому мониторингу и получать сведения в рамках существующих НПА.

| Параметр | Значение |
|----------|----------|
| Наименование | Электронный сервис «Сервисный концентратор Росфинмониторинга» |
| Назначение | Единая точка входа для направления сведений в Росфинмониторинг |
| Поставщик | Федеральная служба по финансовому мониторингу |
| Потребитель | Пользователь Личного кабинета Росфинмониторинга |
| Режим | Асинхронный |
| Адрес | `https://portal.fedsfm.ru:8081/Services/fedsfm-service` |
