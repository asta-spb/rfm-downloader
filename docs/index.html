1.1. Описание электронного сервиса

Электронный сервис «Сервисный концентратор Росфинмониторинга» предоставляет возможность пользователям Личного кабинета на официальном сайте Росфинмониторинга направлять в Федеральную службу по финансовому мониторингу и получать сведения в рамках существующих НПА.

Табл. 2. Описание электронного сервиса
НаименованиеЭлектронный сервис «Сервисный концентратор Росфинмониторинга»
НазначениеЕдиная точка входа для направления сведений в Росфинмониторинг
ПоставщикФедеральная служба по финансовому мониторингу
ПотребительПользователь Личного кабинета Росфинмониторинга
РежимАсинхронный
Адресhttps://portal.fedsfm.ru:8081/Services/fedsfm-service

3. Руководство пользователя

3.1. Варианты значений поля "Обязательность"

Требования к наличию и заполнению полей:

  • О — обязательный элемент (наличие и заполнение)
  • Н — необязательный элемент
  • УО — условно-обязательный элемент. Условия указаны в поле «Особенность»

Незаполнение обязательных полей не допускается!

3.2. Установление https-соединения

При вызове любого метода сервиса для установления защищённого по ГОСТ https-соединения необходимо передавать сведения о сертификате инициатора запроса (тот же сертификат, который используется для входа в защищённый Личный кабинет на официальном сайте Росфинмониторинга).

4. Получение доступа к методам

4.1. Порядок получения доступа к тестовым методам электронного сервиса

Для получения доступа к интересующим тестовым методам необходимо через свой Личный кабинет на официальном сайте Росфинмониторинга направить заявку в техническую поддержку, указав в поле Тематика обращения «Сервисный концентратор», а в тексте указать: «Регламентная процедура. Получение доступа к тестовым методам ЭС».

4.2. Порядок получения доступа к продуктивным методам электронного сервиса

Для получения доступа к продуктивным методам ЭС необходимо:

  1. Успешно пройти тестирование, вызвав все тестовые методы ЭС;
  2. Заполнить форму заявки на получение доступа к продуктивным методам ЭС и вместе с неформатированными конвертами запросов-ответов (требуются только те запросы и ответы, которые содержат параметры) к тестовым методам ЭС в формате JSON упаковать в zip-архив. Прикладывать конверт запрос-ответа метода authenticate не требуется;
  3. Сформированный архив направить в Росфинмониторинг, заполнив заявку в техническую поддержку и указать в Тематике обращения: «Сервисный концентратор», а в тексте заявки: «Регламентная процедура. Получение доступа к продуктивным методам ЭС».
