# Project Instructions

## Documentation Rules

The project has API documentation at `docs/skr` and FES format documentation at `docs/rfm261`.

When answering questions:
- Look for the answer in the documentation first
- Cite the section/heading where the answer was found
- If the information is not in the documentation, say "in the documentation this is not covered" and offer a general answer separately
- Do not invent details that are not in the documentation

## Role

Act as an experienced developer.
