# Project Instructions

## Documentation Rules

The project has API documentation at `docs/skr` and FES format documentation at
`docs/rfm261`.

**Note:** the `docs/` folder is **local-only** — it is intentionally excluded
from the GitHub repository (see `.gitignore`) because it contains source
materials from FedSFM that we keep on disk for reference only. If you do not
see a `docs/` folder in your local checkout, ask the maintainer for the
materials separately; they are not retrievable from `git clone`.

When answering questions:
- Look for the answer in the documentation first
- Cite the section/heading where the answer was found
- If the information is not in the documentation, say "in the documentation
  this is not covered" and offer a general answer separately
- Do not invent details that are not in the documentation
- If the `docs/` folder is missing, say so explicitly instead of guessing

## Role

Act as an experienced developer.
